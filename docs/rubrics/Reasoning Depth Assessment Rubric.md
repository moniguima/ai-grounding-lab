### Logical Fallacy Detection

**Check for reasoning errors:**

- [ ] **Circular reasoning** (conclusion assumes what it's proving)
- [ ] **False dichotomy** (only 2 options when more exist)
- [ ] **Post hoc** (correlation → causation without evidence)
- [ ] **Overgeneralization** (specific → universal without justification)
- [ ] **Equivocation** (shifting meaning of key terms)
- [ ] **Affirming the consequent** (if A→B and B, then A)

```
START → Calculate % of Level 3-4 explanations

├─ YES → Check mechanism explanations
│   └─ All major claims have mechanisms?
│       ├─ YES → Check theories

│       │       ├─ YES → Check logic
│       │       │   └─ No fallacies, coherent flow?

│       │       │       │       ├─ YES → SCORE = 5
```

## Troubleshooting

### Issue: "I can't tell if this is Level 2 or Level 3"

**Solution:**
- **Level 2**: A → B (one step: "X causes Y because Z")
- **Level 3**: A → B → C → D (multiple steps shown)

Count the arrows. If >2 steps, it's Level 3.

---

### Issue: "Is name-dropping a theory enough?"

**Solution:**
No. Theory must be:
1. Explained (what does theory say?)
2. Applied (how does it relate to this claim?)

Just mentioning theory name = Quality 1 (not sufficient for high scores)

---

### Issue: "What counts as a 'mechanism'?"

**Solution:**
Mechanism = Process explanation answering "HOW does X produce Y?"

✓ "X activates Y, which triggers Z, resulting in W"
✗ "X causes W" (no process shown)
