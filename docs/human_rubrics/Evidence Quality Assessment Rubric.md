## STEP 3: Recency Assessment

**Instructions:** Evaluate if citation age is appropriate for the claim type.

### Recency Evaluation Guide

**First, categorize each major claim by topic type:**

| Topic Type | Recency Standard |
|:---|:---|
| **Foundational theory** | Pre-2020 acceptable (e.g., cognitive load theory) |
| **Established methods** | 2015-2020 acceptable (e.g., RLHF basics) |
| **Current AI/LLM** | 2022+ required (e.g., GPT-4 capabilities) |
| **Rapidly evolving** | 2023+ preferred (e.g., AI safety, benchmarks) |
| **Policy/regulation** | 2023+ required (e.g., EU AI Act implementation) |

### **Relevance Level Definitions:**

| Level | Definition | Example |
|:---|:---|:---|
| **DIRECT** | Citation specifically addresses the exact claim | "RAG reduces hallucinations 30-50% (Lewis 2020)" where paper reports this exact finding |
| **INDIRECT** | Citation supports general area but not specific claim | "RAG improves accuracy (Lewis 2020)" where paper shows improvements but different metrics |
| **TANGENTIAL** | Citation relates to topic but doesn't support claim | "LLMs are powerful (Vaswani 2017)" citing Transformers paper |
| **IRRELEVANT** | Citation doesn't meaningfully connect | Using unrelated paper |

## STEP 7: Final Scoring

### Evidence Quality Score (0-5)

**Use the decision tree below:**

```
START HERE ↓

Are there 3+ peer-reviewed citations (Tier 1-3)?
├─ YES → Continue to next check
│   └─ Are most citations COMPLETE (author+year+venue+DOI)?
│       ├─ YES → Continue
│       │   └─ Are citations appropriately RECENT?
│       │       ├─ YES → Continue
│       │       │   └─ Do all major claims have DIRECT citation support?
│       │       │       ├─ YES → Continue
│       │       │       │   └─ Are evidence labels present?
│       │       │       │       ├─ YES → Continue
│       │       │       │       │   └─ Are quantitative metrics provided?
│       │       │       │       │       ├─ YES → SCORE = 5
│       │       │       │       │       └─ NO → SCORE = 4-5
│       │       │       │       └─ NO → SCORE = 4
│       │       │       └─ NO (some claims uncited) → SCORE = 3-4
│       │       └─ NO (outdated citations) → SCORE = 3
│       └─ NO (incomplete citations) → SCORE = 3-4
│
└─ NO → Continue to lower scores
    └─ Are there 1-2 peer-reviewed sources?
        ├─ YES → SCORE = 3 (if reasonably complete)
        └─ NO → Are there ANY verifiable sources?
            ├─ YES → SCORE = 2
            └─ NO → SCORE = 0-1
```

---

### Detailed Scoring Rubric

**SCORE 5: Exemplary** ✓ All criteria met:
- [ ] ≥3 peer-reviewed citations (Tier 1-3)
- [ ] All citations COMPLETE (author+year+venue+DOI)
- [ ] Appropriately recent for topic types
- [ ] All major claims have direct citation support
- [ ] Evidence strength explicitly labeled
- [ ] Quantitative metrics provided
- [ ] Zero uncited factual claims

---

**SCORE 4: Strong** ✓ Most criteria met:
- [ ] 2-3 peer-reviewed citations (Tier 1-4)
- [ ] Most citations include author+year+venue (DOIs may be partial)
- [ ] Recency appropriate for most claims
- [ ] Major claims cited (1-2 minor claims may lack attribution)
- [ ] Some evidence categorization present
- [ ] Some quantitative detail

**What's missing:** May lack DOIs, 1-2 uncited claims, inconsistent evidence labels

---

**SCORE 3: Adequate** ✓ Minimum criteria met:
- [ ] 1-2 peer-reviewed sources OR multiple lower-tier sources (Tier 4-6)
- [ ] Basic citation info (author+year) provided
- [ ] Mix of recent and older sources
- [ ] Some major claims cited; others use generic references
- [ ] No systematic evidence categorization
- [ ] Minimal quantitative detail

**What's missing:** Few high-tier sources, incomplete citations, generic "research shows", no evidence labels

---

**SCORE 2: Weak** ⚠️ Below minimum:
- [ ] 0-1 peer-reviewed sources
- [ ] Heavy reliance on generic references ("studies show")
- [ ] Citations incomplete or unclear
- [ ] Many claims uncited
- [ ] No evidence categorization
- [ ] No quantitative metrics

**Key problem:** Relies on authority without proper attribution

---

**SCORE 1: Poor** ⚠️ Critically deficient:
- [ ] Zero verifiable citations
- [ ] All claims unattributed
- [ ] No way to verify information
- [ ] Opinion or common knowledge only

---

**SCORE 0: No Evidence**
- [ ] Entirely unsupported assertions
- [ ] No citation attempt
